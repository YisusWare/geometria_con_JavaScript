Con esto en mente mostraré a continuación la manera de calcular tanto el área como el perímetro de algunas figuras geométricas, así como los programas con los cuales puedes calcular estas medidas con JavaScript (aclarando que una vez desglosado el algoritmo para calcular dichas medidas lo puedes hacer en cualquier lenguaje de programación).

Calculando las medidas de un cuadrado

Calcular el perímetro de un cuadrado debe ser de las cosas más sencillas de hacer, el perimetro de un cuadrado es dado por la suma de sus lados, o lo que es lo mismo, la multiplicación de uno de sus lados por cuatro.

En el caso del área, podemos calcularla al elevar al cuadrado la medida de uno de los lados del cuadrado, es decir, al multiplicar lado por lado.

Podemos hacer un programa que calcule las medidas de cualquier cuadrado a partir de uno de sus lados utilizando el siguiente código. 

 
                    
                    
   function calcularPerimetroCuadrado(lado){
        return lado * 4
   }
                    
   function calcularAreaCuadrado(lado) {
        return Math.pow(lado,2)
   }

Calculando las medidas de un triangulo

Para calcular las medidas de un triangulo contamos con dos formulas generales, en el caso del perímetro, simplemente tenemos que sumar cada uno de los lados del triángulo, así como hicimos con el cuadrado.

Por otro lado, el área la calculamos multiplicando la base por la altura y dividiendo el resultado entre 2.

Sin embargo, nos encontramos con un inconveniente ya que nosotros deseamos calcular el área a partir de los lados del triángulo, así que ¿Cómo podemos hacer esto?

function calcularPerimetroTriangulo(a,b,c){
    return parseFloat(a) + parseFloat(b) + parseFloat(c)
}


function calcularAreaTriangulo(a,b,c){
    let s= (calcularPerimetroTriangulo(a,b,c)/2)
    
    return Math.sqrt(s * (s-a) * (s-b) * (s-c))
}
