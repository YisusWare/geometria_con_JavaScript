Taller 1: Medidas de figuras geometricas

Al momento de trabajar con figuras geométricas, es esencial para un ingeniero conocer las medidas de estas, casi todo espacio puede ser descrito utilizando estas y conocer sus medidas y ser capaces de computarizar cálculos geométricos debería ser algo sencillo para cualquier ingeniero de software. Con esto en mente mostraré a continuación la manera de calcular tanto el área como el perímetro de algunas figuras geométricas, así como los programas con los cuales puedes calcular estas medidas con JavaScript (aclarando que una vez desglosado el algoritmo para calcular dichas medidas lo puedes hacer en cualquier lenguaje de programación).

Calculando las medidas de un cuadrado

Calcular el perímetro de un cuadrado debe ser de las cosas más sencillas de hacer, el perimetro de un cuadrado es dado por la suma de sus lados, o lo que es lo mismo, la multiplicación de uno de sus lados por cuatro.

En el caso del área, podemos calcularla al elevar al cuadrado la medida de uno de los lados del cuadrado, es decir, al multiplicar lado por lado.

Podemos hacer un programa que calcule las medidas de cualquier cuadrado a partir de uno de sus lados utilizando el siguiente código. 

 
                    
                    
    function calcularPerimetroCuadrado(lado){
        return lado * 4
    }
                    
    function calcularAreaCuadrado(lado) {
        return Math.pow(lado,2)
    }
